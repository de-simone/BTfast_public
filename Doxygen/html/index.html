BTfast
BTfast: Back-Test fast!

Introduction

BTfast is a software written in C++ to do event-driven backtesting and optimization of trading systems.

Version
v1.0
Author
Andrea De Simone

Storage

CSV files, SQLite database

Required files:

  • Configuration file: 'settings.xml'
  • Data file, in BarData/ directory
  • Strategy files: 'Strategy/strategyname.cpp', 'Strategy/strategyname.h', 'Strategy/StrategyName.xml'

EDIT:

  • Input settings (in 'settings.xml')
  • Strategy parameters (in 'Strategies/StrategyName.xml')
