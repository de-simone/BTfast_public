Detailed Description

Class providing real-time status and handling of open positions. If there are open positions, it updates the account class on every incoming bar.

Member Variables

  • account_: reference to instance of account class, initialized by constructor when instantiating BTfast object.
  • events_queue_: pointer to events_queue, initialized in BTfast.run_backtest() and set via method.
  • open_positions_: vector of open positions (instances of Position class)
