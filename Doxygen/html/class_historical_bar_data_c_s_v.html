Detailed Description

Read historical bars from a single CSV file, store them into vector 'bar_data_' and stream them to the provided events queue.

Member Variables:

  • csv_format_: option to deal with different formats of input CSV file 1 = intraday data exported from TradeStation 2 = daily data exported from TradeStation 3 = data exported from MatLab
  • csv_file_: name of csv file
  • symbol_: object of instrument class
  • timeframe_: timeframe
  • start_date_: start date included (from settings)
  • end_date_: end date excluded (from settings)
  • events_queue_: pointer to events_queue, initialized in BTfast.run_backtest() and set via method.
  • continue_parsing_: switch to control parsing
  • bar_data_: vector containing all bars over selected date range
  • iter_data_: iterator over bar_data_ vector
  • tot_bars_: total number of bars to parse
  • infile_: pointer to a FILE object that identifies the stream
  • db_: pointer to SQLite database
  • stmt_: SQLite statement
